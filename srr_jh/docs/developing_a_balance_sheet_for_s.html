Developing a Balance Sheet for Sidewalk Rebate Program and assigning new applications to wait-list:

Assuming the Starting Balance  =   $6,000,000

Count the Number of Application Received (not reviewed by BPW at this point):

For each application received a $2000 or $4000 will be assigned as RESERVED_AMT to this application.

Example:

 24 Residential Property Applications
15 Commercial Property Applications

 Reserved funds = 24 x $2000 + 15 x $4000 = $108, 000

Once an offer is made, an OFFERED_AMT will be assigned to each of those applications and the RESERVED_AMT will be set to 0.  This means that any application at any point of time has either a Reserved Amount or Offered Amount and for "Not Eligible", "Declined", etc.  both amouts will be 0.

At any point of time the AVAILABLE FUND = Starting Balance - Total of All Reserved Amounts - Total of All Offered Amounts

The application OFFERED_AMT to 0 if applicant declined offer or offer expired so that the running total will be accurate.

The available funds will be calcuated each time a new application is received and at a certain amount, all new applications will be assigned a status of "Wait-Listed".

Another algorithm will be developed to move applications from "Wait-Listed" to the "Pending Review" queue whenever an application is deemed "Not Eligible", "Declined", etc..
There could be a time gap between moving applications from "Wait-Listed" to "Pending Review" queue based on time allowed for appeal.
